Django CRM

Fully functioning CRM tool.

Features

Authentication

The project uses a custom user model to authenticate users. Users can signup, login and logout.

CRUD

Users can create, read, update and delete leads.

PostgreSQL

The project uses PostgreSQL for the database, hosted on digital ocean.

Password Reset

The project features fully functioning password reset via email sending.

Django Signals

The project uses Django signals in order to create user profiles the moment a new user signs up.

Class Based Views

The project uses function based views and class based views.

Leads & Agents

Apart from leads, users with the correct permissions can create and manage agents. Agents belong to organizations.

User Roles

Distinct user roles manage what users can see and do. There are agents and organizations.
